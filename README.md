## <span id="1">1. Concept 概念/术语 </span>

- PET : Platform Engineering Team 平台工程团队
- IDP : Internal Developer Platform 内部开发者平台 
- DX/DevX : Developer Experience (also EX: Engineer Experience) 开发者体验/工程师体验
- self-service API 自服务API
- DevOps 开发运维一体化
- SRE 站点可靠性工程
- Cloud Native 云原生
